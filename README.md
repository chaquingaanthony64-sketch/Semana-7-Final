-Nombre del proyecto:

Generador de contraseñas 
-Integrantes:

Anthony Mauricio Chaquinga Freire

-Objetivo del sistema:

El objetivo de un generador de contraseñas es crear claves seguras automáticamente para proteger cuentas y cualquier información personal.
Lo que busca es ayudar a no usar contraseñas fáciles de adivinar como mi nombre, mi fecha de nacimiento o números simples.

En su lugar, genera combinaciones más seguras con letras, números y símbolos que son difíciles de descifrar.

Gracias a esto, se puede mantener redes sociales, correos y otras cuentas más protegidas, evitando que otras personas puedan acceder a la informacion. 

-Descripción de funcionalidades 

El programa primero permite al usuario definir la longitud de la contraseña, es decir, cuántos caracteres quiere que tenga la clave. 

Luego le da la opción de elegir qué tipo de caracteres desea incluir: mayúsculas, minúsculas, números y símbolos.

Según las opciones que el usuario selecciona, el sistema va formando una “lista” de caracteres posibles para crear la contraseña. 

Si el usuario no selecciona ningún tipo de carácter, el programa muestra un mensaje indicando que debe elegir al menos uno.

Después, el sistema genera la contraseña de forma automática. Para hacerlo, toma caracteres al azar de los que fueron seleccionados y los va juntando hasta completar la longitud indicada.

Una vez creada la contraseña, el programa la muestra en pantalla y luego pregunta si el usuario desea guardarla. Dependiendo de la respuesta, el sistema confirma si fue guardada o no.

Finalmente, el programa termina mostrando un mensaje de finalización.

En resumen, este sistema permite:

Elegir el tipo de caracteres

Definir la longitud de la contraseña

Generar una contraseña aleatoria

Mostrar la contraseña creada

Decidir si se guarda o no

Finalizar el proceso de forma ordenada
